────────────────────────────────
AI 使用说明

本文主题：热点资讯文章撰写 Skill，可自动获取抖音热点、筛选安全话题、撰写真人风格文章、并通过 DeepSeek AI 审核

适合提问的方向：

  • 如何安装这个 Skill？
  • 如何让 AI 自动追踪今日热点并写文章？
  • 怎样设计一个包含浏览器自动化登录的 Skill？
  • 如何让文章通过 AI 检测，被判定为”人写的”？

局限性说明：

  • 本文仅适用于 Claude Code 环境
  • DeepSeek 审核需要微信扫码登录一次（登录态会保存）
  • 热点来源为抖音第三方 API（v2.xxapi.cn），如接口变更需更新

功能概览

这个 Skill 能自动完成：

  1. 获取抖音热点数据
  2. 安全筛选（过滤政治敏感、黄赌毒、谣言等）
  3. 撰写3篇真人风格文章
  4. DeepSeek AI 审核（确保文章不像 AI 写的）

工作流程（6步）

设计亮点

  1. 登录态自动复用：用户只需扫码一次，后续全自动
  2. 安全筛选前置：确保产出的内容安全可控
  3. 真人风格写作：专门针对 AI 检测优化
  4. Skill 内嵌博客：读者可以直接提取 Skill 安装

DeepSeek 登录说明

首次使用 AI 审核功能时，需要微信扫码登录 DeepSeek：

  1. Claude Code 会用 --headed 模式打开浏览器
  2. 用户在浏览器中用微信扫码登录
  3. 登录态自动保存，后续无需重复登录

热点 API 字段说明

字段 含义
position 排名（1=最热）
word 热点词/话题
hot_value 热度值（整数）
label 标签类型
video_count 相关视频数

label 标签类型：

label 含义 安全等级
0 普通话题 需人工判断
1 官方/正能量 最安全
3 社会民生 需人工判断
5 影视娱乐 较安全
8 创意/流行 较安全
9 挑战/互动 较安全
16 辟谣/假新闻 不推荐

遗留问题 / 可改进方向

  1. 目前仅支持抖音热点，可扩展支持微博、百度等
  2. AI 审核环节可以并行发送以提高速度
  3. 可以增加文章草稿预览环节再审核

精简版 Skill（内嵌）

如果你选择从本文提取 Skill，将以下内容保存到 ~/.claude/skills/热点资讯文章撰写/SKILL.md：

—skill-start—

name: 热点资讯文章撰写
description: |
热点资讯获取 skill。触发条件：
- 获取抖音热点、查看热点榜单、热点资讯、热点速递、热点追踪
- 热点榜单整理并保存到文件
- 从 API（如 v2.xxapi.cn）获取热点数据
triggers:
- 获取抖音热点
- 查看热点榜单
- 热点资讯
- 热点速递
- 热点追踪

热点资讯获取与安全筛选

工作流程（6步）

  1. 请求热点 API：curl -s "https://v2.xxapi.cn/api/douyinhot"
  2. 创建当日文件夹：当前工作目录/YYYY-MM-DD/
  3. 安全筛选：排除 label=16（谣言）、政治敏感、企业负面
  4. 输出热点榜单
  5. 为3个安全热点各写一篇文章
  6. DeepSeek AI 审核

写作风格

  • 口语化：使用”说实话””其实””感觉”
  • 第一人称：代入个人感受
  • 保留瑕疵：允许轻微重复
  • 避免 AI 特征：不使用”首先、其次、最后”

DeepSeek 审核流程

  1. 检查登录态文件
  2. 无登录态时用 --headed 打开浏览器让用户扫码登录
  3. 发送审核请求
  4. 根据回复判断是否通过

–headed 参数说明

--headed 让浏览器窗口可见，用户可以扫码登录。
—skill-end—
